# 🎓 Расписание университета
Веб-приложение для просмотра расписания занятий в учебном заведении.

## 🏗️ Технологический стек
- ⚡ FastAPI – высокопроизводительный Python-фреймворк для создания API
    - 📊 SQLALchemy – ORM для взаимодействия с базами данных SQL
    - ✅ Pydantic – валидация данных и управление настройками
    - 🐘 PostgreSQL – реляционная база данных для хранения основной информации
    - 🟥 Redis – система кэширования и управления сессиями
- ⚛️ React – библиотека для построения интерактивного пользовательского интерфейса
    - 🎨 DaisyUI – компоненты на основе Tailwind CSS
    - 📝 React Query - управление состоянием и кэширование
    - 📅 Tailwind CSS - утилитарные стили
    - 📩 Axios - HTTP-клиент для взаимодействия с API
- 🔒 JWT-аутентификация – аутентификация с использованием JSON Web Tokens
- 🐳 Docker Compose – контейнеризация для разработки и продакшена
- 🧪 Тестирование с Pytest – тестирование кодовой базы

## Краткое описание
Это веб-приложение представляет собой готовый шаблон для создания системы управления и просмотра расписания в учебных заведениях (университетах, колледжах, школах).

Приложение позволяет:

* удобно формировать и редактировать расписание занятий;
* просматривать расписание по группам, преподавателям или аудиториям;
* легко редактировать и уведомлять пользователей об изменении в рассписании

Особенностью проекта является встроенная интеграция с Telegram-ботом, реализованным внутри этой же системы (без использования сторонних сервисов). Бот позволяет пользователям получать уведомления об изменениях в расписании в реальном времени.

## 🚀 Быстрый старт
### Клонирование репозитория

```bash
git clone https://github.com/actiotrue/shcedule-university.git
cd shcedule-university
```

### Установка зависимостей через uv
```bash
cd backend
uv venv
uv sync
```

### Запуск delevlopment среды
```bash
cd backend
uv run uvicorn app.main:app
cd ../fronntend
npm run dev
```

### Применение миграций
```bash
uv run alembic upgrade head
```

### Сборка и запуск всех сервисов
```bash
docker-compose up -d --build
```

### Тестирование
```bash
cd backend
pytest
```
